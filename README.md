# EKS creation helper
This is a script repository for create EKS in easy way.

## Simple Steps
#### 1. Edit the NAMING_CONF
```
vim NAMING_CONF
```
#### 2. Create Roles and Policies
```
./create_eks_roles.sh
```
#### 3. Create EKS Cluster
```
./create_eks_cluster.sh
```
#### 4. Create EKS Node Group
```
./create_eks_nodegroup.sh
```
